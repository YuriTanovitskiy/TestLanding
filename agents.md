# AGENTS.md

## Project

Auto CRAFT KZ landing page.

## Stack

- React
- TypeScript
- Vite

## Rules

- Mobile First.
- Responsive layout.
- Semantic HTML.
- Functional React components only.
- Use TypeScript interfaces for props.
- Avoid `any`.
- Keep components focused and reusable.
- Do not modify unrelated files.
- Do not install new dependencies unless necessary and explain why.
- After significant changes always run:

```bash
npm run build
```

- If the task affects UI, verify that it works on:
  - 375 px
  - 768 px
  - 1440 px

- Preserve existing project structure unless instructed otherwise.
