# Work content v1

Рабочая, отобранная версия контента для лендинга Auto CRAFT KZ.

## Источник

- `../content/` — неизменяемый RAW-контент.
- Файлы в этой папке могут быть отобраны, переименованы, обработаны и оптимизированы для сайта.
- RAW-файлы не удаляются и не перезаписываются.

## Правила подготовки

1. В рабочую версию переносится только материал, имеющий понятное назначение на странице.
2. Для каждого файла фиксируется источник и характер обработки.
3. Дубликаты и слабые варианты не переносятся.
4. Фотографии кейсов группируются по схеме: автомобиль → симптом → причина → выполненная работа → результат.
5. Перед подключением к сайту изображения подготавливаются в адаптивных размерах и современных форматах.
6. Генеративно обработанные изображения явно маркируются и требуют визуального подтверждения владельца.

## Текущее состояние

- Создана рабочая структура v1.

## Обработка фотографий сотрудников

Активные версии подготовлены без генеративного редактирования. Выполнено центральное кадрирование до единого формата 4:5 и сохранение в JPEG с качеством 88. Пиксели лиц и окружения не реконструировались.
